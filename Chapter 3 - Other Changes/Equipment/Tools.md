**_Tool Training_**: Similar to other equipment, tools now work through Training.

**_Proficiency Conversion_**: Features that previously gave Proficiency with a set of tools now gives Training with those tools instead. When a feature previously required Proficiency with a set of tools, it now requires Training with this set of tools instead.

**_Utilizing Tools_**: When using the Utilize option of a set of tools, make the ability check with the given ability and choose a fitting skill. That skill determines whether your Proficiency Bonus is added to the roll or not.

**_Tools as Requirements_**: Some ability checks require the use of tools to be rolled at all. For example, it is not possible to use the Craft options of a set of tools without using the tools and having Training with those tools.

**_Improvised Tools_**: When you want to use a set of tools you are trained with but don't have the necessary tools on hand, you may use objects similar to the required tools instead. In that case, you roll related ability checks with Disadvantage, and the needed time to craft an item is doubled.

**_Flexible DCs_**: Instead of using the fixed DC given in the base rules, some applications for the Utilize options of a set of tools may call for different DCs. For example, some locks are more difficult to pick than others. In those cases, adjust the DC accordingly.
#### Surgery Tools
Surgery Tools are a new type of Artisan's Tools that are used in [[Surgery]] and [[Grafting]]. Cost: 30 GP. Ability: Intelligence. Skills: Sleight of Hand, Nature, Medicine. Weight: 4 lb. Utilize: [[Surgery|Treat disabled body part]], [[Grafting]].
